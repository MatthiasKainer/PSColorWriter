# PSColorWriter

Write colored lines to the command line. 

## Usage

        Write-Output-Color -Red "Alert!"
		Write-Output-Color -Green "All fine"
		Write-Output-Color -Red "Alert!" -White "This is not a test"
		
## Colors

* -Black
* -DarkBlue
* -DarkGreen
* -DarkCyan
* -DarkRed
* -DarkMagenta
* -DarkYellow
* -Gray
* -Darkgray
* -Blue
* -Green
* -Cyan
* -Red
* -Magenta
* -Yellow
* -White
